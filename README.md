# Rails Forms Tic-Tac-Toe

A Tic-Tac-Toe game powered by Rails forms. 

## Prerequisites
Ruby and Bundler gem

## Setup
```sh
bundle install
rake db:migrate
rails s 
```

The app should be accessible through `localhost:3000` in your browser.
